1. Регистрация пользователя
POST /users/register
Тело: { "email": "...", "password": "...", "password2": "...", "first_name": "...", "last_name": "..." }

2. Профиль пользователя
GET /users/profile — получить профиль
PATCH /users/profile — редактировать профиль: { "first_name": "...", "last_name": "..." }

3. JWT-токены
POST /api/token/ — получить токены: { "email": "...", "password": "..." }
POST /api/token/refresh/ — обновить access token: { "refresh": "<refresh_token>" }
POST /users/logout/ — выйти: { "refresh": "<refresh_token>" }

4. Онбординг организации

4.1 Выбор типа организации
PATCH /organization/profile
Тело: { "org_type": "ie" } или "llc"

4.2 Выбор налогового режима
PATCH /organization/profile
Тело: { "tax_regime": "single" } или "general"

4.3 Добавление видов деятельности + ставок
POST /organization/activities
Тело: { "activity": <id>, "cash_tax_rate": "4.00", "non_cash_tax_rate": "2.00", "is_primary": true }

4.4 Редактирование / удаление вида деятельности
PATCH / DELETE /organization/activities/<id>/

4.5 Завершение онбординга
POST /organization/profile/finalize
→ onboarding_status = "completed"

4.6 Проверка статуса онбординга
GET /organization/status
Ответ: { "onboarding_status": "org_type", "is_completed": false }

5. Финансы

5.1 Категории
GET /finance/categories/ — список категорий (системные + пользовательские)
POST /finance/categories/ — создать категорию: { "name": "...", "category_type": "income" }
PATCH / DELETE /finance/categories/<id>/

5.2 Транзакции
GET /finance/transactions/ — список транзакций пользователя
POST /finance/transactions/ — добавить транзакцию: { "amount": "1000.00", "transaction_type": "income", "category": 5, "description": "...", "transaction_date": "2026-02-09", "payment_method": "cash", "is_business": true, "is_taxable": true, "activity_code": 123 }
PATCH / DELETE /finance/transactions/<id>/

5.3 Сводка по категориям
GET /finance/transactions/summary_by_category/
Поддерживаются фильтры: date_from, date_to, transaction_type, category, is_business, payment_method, is_taxable, activity_code
